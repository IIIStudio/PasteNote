需求：创建类似 memos 的自托管笔记和知识管理浏览器插件

核心目标：轻量、快速、隐私优先的本地笔记工具，支持可选云同步

设计风格：
- 黑白配色为主，辅以绿色（贡献日历）等强调色
- 扁平化设计，无阴影、渐变，突出简洁与可读性
- 界面元素以矩形、圆形为主，线条利落，间距统一
- 布局紧凑，减少冗余留白，信息密度适中
- 去 AI 风格，避免圆角过度、渐变装饰，采用硬朗几何感与实用主义视觉

功能需求：
- 本地 JSON 存储（默认）
- 可选腾讯云 COS S3 同步（参考：https://cloud.tencent.com/document/product/436/11459）
- 本地支持 JSON 文件导入/导出
- GitHub 风格贡献日历（绿墙）可视化笔记活跃度
- 标签系统（支持多标签分类、筛选）
- 全文搜索功能（实时匹配标题/内容/标签）
- 内容点击复制（选中文本一键复制，带反馈提示）
- 浏览器插件形态（Manifest V3），最终输出压缩包
- 点击插件图标时，在页面居中显示页面弹窗（模态框形式，背景遮罩可选）

技术提示：
- 腾讯云 COS SDK 建议本地引入（下载 cos-js-sdk-v5.min.js：https://github.com/tencentyun/cos-js-sdk-v5/blob/master/dist/cos-js-sdk-v5.min.js）
- 优先保证本地功能稳定性，云同步作为可选增强
- 插件需兼容主流浏览器（Chrome/Edge/Firefox）
- 界面简洁，操作直观，加载速度快
